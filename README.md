# unattended-setups
A set of unattended setup tools for Debian-based Distros.

## Supported tools
| Tool   | Command                                                       |
|--------|---------------------------------------------------------------|
| Docker | sh -c "$(curl -fsSL https://thmr.at/setup/docker)"   |
| ZSH    | sh -c "$(curl -fsSL https://thmr.at/setup/zsh)"      |
